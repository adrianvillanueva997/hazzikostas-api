# hazzikostas-api
Api done for the Discord bot Hazzikostas. Backend done in Golang and a small frontend application done in React.
Currently, the API has a cronjob to update the characters every 6 hours.

Technologies used:
 * Golang (gomodules)
   * Gingonic
   * Godotenv
   * Gocron
 * Javascript
   * React
